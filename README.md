## About

I am a Java backend developer focused on building systems that solve real problems. I care about the quality of what I ship, the impact it creates, and the discipline it takes to build it well.

My work spans backend engineering, distributed systems, and digital public infrastructure. I contribute to open source because I believe software, when built with intent, can create lasting and meaningful change.

I approach engineering the same way I approach growth — with patience, curiosity, and a long-term perspective.
